Все вычисления - calculation.ipynb
37Ar_Bahcall_sigma.csv - значения сечения захвата нейтрино для 37Ar из статьи Бакала: Bahcall et al., Phys. Rev. C 54, 411 (1996); arXiv:nucl-th/9601044
j_data_37Ar_minimal_values.csv - значения J_k для 37Ar; для состояний с неизвестным J_k считалось что J_k=0.5

j_data_76As_maximal_values.csv - значения J_k для 76As; для состояний с неизвестным J_k считалось что J_k=1
